
#PDF TO AUDIOBOOKрҹ”ү

A very simple python application which converts a PDF file (text) to Audio via Text-To-Speech libraries. Users can select their preferred PDF file thru the file dialog and can hear them through the application.

---

#FEATURESрҹ’Һ

-> Any PDF file can be selected.
-> PDF can be converted into speech.
-> Does not require internet connection. Can be accessed while being offline.
-> Reads the entire PDF file.

---

#PYTHON LIBRARIES USEDвң’пёҸ

-> pyttsx3 (To convert Text to Speech)
-> PyPDF2 (To access through the contents of the selected PDF file)
-> tkinter

#LIMITATIONSвҡ пёҸ

1. Works only with text based PDFs.
2. The application does not have any currently provide any playback control yet. So, in order to stop playing the audio, press Ctrl+C in the terminal.
